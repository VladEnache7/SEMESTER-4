#### Part I: Client-Side (Frontend)
*   **Structure:** HTML5 Semantic tags (`<article>`, `<section>`, `<nav>`).
*   **Style:** CSS3, Responsive Web Design (Media Queries), Web Fonts.
*   **Logic:**
    *   **DOM:** Document Object Model manipulation (`document.getElementById`).
    *   **Async:** Promises, `setTimeout`, and AJAX (`XMLHttpRequest` / `fetch`).
    *   **Libraries:** jQuery (legacy support and understanding).

#### Part II: Data Exchange
*   **XML:** The strict markup standard, transformed via XSLT.
*   **JSON:** The modern standard for API data transfer.

#### Part III: Server-Side (Backend) Tour
*   **PHP:** Scripting language, easy deployment, loose typing.
*   **Java EE:** Servlets (Java classes handling requests) and JSP (HTML with embedded Java).
*   **ASP.NET:** The Microsoft stack, integrating with C# and the .NET framework.

#### Part IV: Security & Advanced
*   **Security:**
    *   **XSS (Cross-Site Scripting):** Preventing malicious script injection.
    *   **SQL Injection:** Sanitizing inputs to protect databases.
    *   **SOP:** The Same-Origin Principle.
*   **Real-time:** WebRTC (Real-Time Communication).
